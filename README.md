# Padaria do Bairro 🧁

Website da Padaria do Bairro - Tradição desde 1970

## Descrição

Site institucional da Padaria do Bairro, desenvolvido para apresentar os produtos, ofertas e informações de contato. O projeto inclui funcionalidades de pesquisa de produtos, cardápio interativo e design responsivo.

## Páginas

- **index.html** - Página inicial com história, ofertas do dia, cardápio e localização
- **ofertas.html** - Página com ofertas semanais
- **contato.html** - Formulário de contato com sistema de avaliação
- **privacidade.html** - Política de privacidade

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)
- Google Fonts

## Como executar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
